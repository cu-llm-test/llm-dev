課題：ブラウザで https://www.yahoo.co.jp というURLにアクセスするとします。このとき、Webページが表示されるまでの過程を時系列に説明してください。ただし、下記に指定した用語をすべて使用してください。

    ブラウザ
    DNS
    IPアドレス
    Webサーバ
    HTTPリクエスト
    HTTPレスポンス

回答：

ブラウザで指定のWebページにアクセスするとき、まず、URL からホスト名をパースし、DNSに問い合わせてIPアドレスを取得する。得られた IP アドレスを使い、Web サーバに接続、HTTP リクエストを送信する。
Web サーバは、受け取った HTTP リクエストの中身を解析し、要求されたページを生成する。そして、そのデータをHTTP レスポンスとして返す。
ブラウザは受け取った HTML レスポンスをレスポンスを解析し、HTML/CSS, Javascript、画像などを読み込む。それらの内容を元に、レンダリングを行うことで、Web ページは表示される。
